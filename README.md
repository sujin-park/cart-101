## Prerequisites

- [Node.js](https://nodejs.org/ko/)
- [Yarn](https://yarnpkg.com/)

## Installation

### 1. Clone the project

```bash
$ git clone https://github.com/sujin-park/cart-101.git
```

### 2. Install Packages

```bash
$ yarn (install)
```

### 3. Start Application

```bash
$ yarn start
```
